# Mermaid Standards

> Mermaid -- Diagrams as Code -- A Markdown-friendly diagramming syntax for flowcharts, sequences, and more.

**Mermaid** is used for technical documentation diagrams in this project. Prefer sequence diagrams over complex graphs; keep diagrams narrow for GitHub rendering.

## Conventions

- **Dark theme**: Use `%%{init: {'theme': 'dark'}}%%` directive
- **Width**: Target 600px max; use short labels and abbreviations
- **Diagram type**: Sequence diagrams preferred over flowcharts/graphs
- **Complexity**: Split large diagrams; avoid crossing lines

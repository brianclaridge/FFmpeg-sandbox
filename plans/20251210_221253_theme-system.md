# Theme System Implementation

## Problem
- Slider track invisible (same color as container background)
- Current theme aesthetically lacking
- No theme switching capability

## Solution

### 1. Fix Slider Visibility
- Add explicit track styling with contrasting color
- Ensure cross-browser compatibility (webkit + moz)

### 2. Create Darcula Theme
Based on JetBrains Darcula palette:
- Background: #2b2b2b (main), #3c3f41 (secondary), #313335 (cards)
- Text: #a9b7c6 (primary), #808080 (secondary)
- Accent: #cc7832 (orange), #6897bb (blue links)
- Success: #6a8759, Error: #ff6b68

### 3. Add Theme Dropdown
- Dropdown in header for theme selection
- Themes: Default (current pink), Darcula
- Persist selection in localStorage
- Apply via CSS class on body element
